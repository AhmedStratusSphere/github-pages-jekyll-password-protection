R-IT Set Cost Code on Standard Cost Code

This NetSuite User Event Script sets the cost code on the line items of journal entries and credit card charges based on the standard cost code and project.

@NApiVersion

2.1

@NScriptType

UserEventScript

@Title

R-IT Set Cost Code on Standard Cost Code

@module

r_it_ue_set_cc_on_stcc

@requires

  • N/log
  • N/search
  • N/record
  • N/runtime

Functionality

The script executes during the afterSubmit event and performs the following actions:

  1. Loads the new record submitted.
  2. Checks if the record type is journalentry or creditcardcharge.
  3. Retrieves the total number of lines in the record.
  4. Collects all unique projects and standard cost codes from the line items.
  5. Retrieves the internal IDs of the standard cost codes.
  6. Retrieves the cost codes based on the project and standard cost code.
  7. Sets the cost code on the line items based on the retrieved cost codes.
